## 網站結構

| 檔案 | 用途 |
|---|---|
| `index.html` | 首頁 |
| `resources.html` | 免費資源（Lead Magnet） |
| `notes.html` | 內容筆記列表 |
| `note-detail.html` | 單篇文章範本 |
| `products.html` | 產品（299 / 1999） |
| `services.html` | 高價代做服務 |
| `case-studies.html` | 案例展示 |
| `partners.html` | 合作分潤 |
| `industry-packs.html` | 產業授權包 |
| `checkout.html` | 人工付款頁 |

- 主色：深海軍藍 `#061426`
- 輔色：冷銀灰 `#C5CDD9`

## 部署

Vercel + GitHub。每次 push 到 `main` 自動部署。

詳見 `DEPLOY.md`（私人文件，不在 repo 中）。

## 商業模式

4 層產品階梯：
- 免費：10 種模型表（收名單）
- 入門：NT$ 299 啟動筆記
- 完整：NT$ 1,999 模板包
- 代做：NT$ 4,980+ 雛形代做包
- 授權：NT$ 19,800+/年 產業授權包

## 表單後端

所有表單透過 [Formspree](https://formspree.io) 收件。每個表單對應獨立 endpoint。
